
# SO Rawan Website

Website untuk input data Stock Opname barang rawan hilang yang terintegrasi dengan Google Sheets dan WhatsApp Bot.

## Fitur

- Input data SO barang rawan hilang
- Terintegrasi dengan Google Sheets
- Sinkronisasi dengan WhatsApp Bot
- Multi-input support
- Riwayat input real-time
